HTML introduction

What is HTML?

HTML is the standard markup language for creating web pages.
HTML stands for Hyper Text Markup Language
It consists of a series of elements represented by tags which tell the browser how to display content.

Example

<div class="html">
    <html>
      <head>
        <title>Page title</title>
      </head>
      
      <body>
        <p>A paragraph</p>
      </body>
    </html>
</div>
